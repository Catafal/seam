Intent

Add release-polish browser coverage for the Topology/3D Explorer path. The checks prove that the real built SPA and Python server can render a nonblank WebGL scene, survive selection/reset, and show a visible error when layout data fails.

Decisions

  • Real server harness: Playwright starts seam serve on a deterministic temporary fixture instead of mocking API responses.
  • Pixel invariants over snapshots: PNG metrics catch blank/white-out canvases without brittle GPU-specific image diffs.
  • URL-gated visual QA hook: node selection uses ?seam-visual-qa=1 so tests do not depend on raycast coordinates that vary by GPU backend.
  • Optional workflow: visual QA runs in a dedicated workflow for web/layout changes rather than slowing the core Python gate.
