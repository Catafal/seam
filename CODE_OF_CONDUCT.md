# Code of Conduct

## Our commitment

Seam is a welcoming project. Everyone who participates — through issues, pull requests,
discussions, or any other channel — is expected to treat others with respect.

## Expected behaviour

- Use clear, constructive language.
- Acknowledge that others may have different experience levels, backgrounds, or goals.
- Give and receive feedback graciously; focus on the work, not the person.
- Accept editorial decisions without extended argument.

## Unacceptable behaviour

The following are grounds for removal from the project:

- Personal attacks or sustained hostility toward any participant.
- Publishing private information (contact details, location) without consent.
- Any conduct that a reasonable person would consider inappropriate for a professional
  open-source setting.

## Scope

This code of conduct applies to all project spaces — GitHub issues, pull requests, discussion
threads, and any community channel where project business is conducted.

## Enforcement

Concerns can be reported privately to **aurasystemsyt@gmail.com**.  
All reports are reviewed promptly and kept confidential.

Responses range from a private request to modify behaviour, through a temporary suspension of
participation, to a permanent ban — depending on severity and pattern.
